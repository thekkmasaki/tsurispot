# 03. アクセスと検索の見方（GA4・サチコ）

> 「数字の読み方」。どの画面で何を見て、良い/悪いをどう判断し、悪いとき何を疑うか。まず「見る習慣」をつけるのが目的。

最終更新: 2026-07-07

---

## 📊 2つの計測ツール
| ツール | わかること | ひとこと |
|---|---|---|
| **GA4（Googleアナリティクス）** | 来た人の数・行動 | 「来たあと」を見る |
| **サチコ（Google Search Console）** | Google検索での見え方・順位 | 「来る前（検索）」を見る |

両方Googleアカウントでログイン。

---

## 1️⃣ GA4（アクセス解析）
- URL：https://analytics.google.com
- 計測ID：`G-FD4XGHP9FN`

ログイン後の「ホーム」でPVの増減の雰囲気をつかめます。

| 数字 | 意味 | 見方 |
|---|---|---|
| **表示回数（PV）** | ページが見られた回数 | 一番大事。増加ならOK |
| ユーザー/セッション | 訪問者/訪問回数 | PVと一緒に増えれば健全 |
| エンゲージメント時間 | 滞在時間 | 長いほど良い記事 |
| 参照元/メディア | どこから来たか | `google/organic`（検索）が多いのが理想 |

**メモ（注釈）機能**：管理→データの表示→メモ で、グラフに付箋を貼れます。大きな変更や障害のとき残すと、あとで数字の変化の理由がわかります。

---

## 2️⃣ サチコ（Search Console）
- URL：https://search.google.com/search-console
- 対象：`tsurispot.com`

| 項目 | 意味 | 狙い |
|---|---|---|
| クリック数 | 検索から実際に来た数 | 増やす |
| 表示回数 | 検索結果に出た数 | 多いのにクリック少→タイトル改善余地 |
| 平均掲載順位 | 検索の平均順位 | 上（数字が小さい）ほど良い |
| インデックス数 | 登録されたページ数 | 実ページ数に近いのが理想 |

### 💎 「あと少しで上位」は宝の山
検索順位 **11〜20位** のキーワードは「あと少しで1ページ目（＝アクセス激増）」。ここを改善するのがSEOで最も効率的。専用スクリプトで自動抽出できます（技術作業）：
```
node scripts/metrics/extract-striking-distance.mjs
```

## 🔍 数字が悪化したとき「何を疑うか」
| 症状 | 疑うこと |
|---|---|
| **PVが急落** | ①検索アルゴリズム更新 ②インデックスから消えた ③サイト障害 |
| **PVが不自然に急増（収益は増えない）** | **ボットの偽アクセス**（→ [05章](./05-セキュリティとボット対策.md)） |
| **収益（RPM）が落ちた** | 広告配置・ポリシー違反・季節要因 |
| **インデックス数が減った** | 薄いページ量産で「クロール済み未登録」が増加 |
| **表示は多いのにクリックされない** | タイトル・説明文が魅力不足 |

> 💡 **PV急増＝喜ぶは早い。** 収益が連動していなければボットを疑う。

## 📅 見る頻度の目安
| ツール | 頻度 | 何を |
|---|---|---|
| AdSense | 週1 | 収益・ポリシーセンター |
| GA4 | 週1 | PVの推移 |
| サチコ | 2週に1 | 順位・「あと少し」・インデックス数 |

毎日は不要。**週1回まとめてチェック**で十分。

---

## 用語ミニ辞典
- **GA4**：来た人の数・行動を測る。
- **サチコ**：Search Consoleの俗称。検索での見え方を測る。
- **オーガニック検索**：広告でない自然な検索流入。理想的。
- **インデックス**：Googleがページを登録すること。されないと検索に出ない。
