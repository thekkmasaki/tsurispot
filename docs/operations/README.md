## 🚦 まず覚える「これだけ」

1. **サイトは自動で動いています。** 毎日張り付く必要はありません。
2. **お金の流れ**：広告（AdSense）とアフィリエイトで**稼ぎ**、AWSサーバー代（月約1.8万円）が**出費**。
3. **一番大事な数字**：`月間PV`。増えると収益もサイト価値も上がる。ゴールは月50万PV。
4. **困ったら [09 困ったとき](./09-困ったとき.md)** を開く。
5. **お金・広告まわりは慎重に**（各章の「⚠️ 禁止事項」を守る）。
