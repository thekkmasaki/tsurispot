# ブログ記事 keyword ターゲット (Phase 3)

AdSense 単体 10x プラン Phase 3 の AI コンテンツ大量生成で狙う long-tail keyword。
6ヶ月で 200 ターゲット × 月 1,500 PV = 30 万 PV/月 を目指す。

## 使い方

1. このリストから 1 keyword 選ぶ
2. `npx tsx scripts/blog-generator.ts "<keyword>" --dry-run` で生成内容を確認
3. 問題なければ `npx tsx scripts/blog-generator.ts "<keyword>"` で microCMS に draft POST
4. microCMS 管理画面で校正 (ファクト確認、 内部リンク追加) → publish
