# ツリスポ Instagram リール用 Geminiプロンプト集

## 使い方
1. Geminiチャット（Pro版）にプロンプトをコピペ
2. 生成された動画をダウンロード
3. 必要ならffmpegでテキスト追加（後述のコマンド参照）
4. Instagramに投稿

## 動画の基本指定（全プロンプト共通で末尾に付ける）
```
【動画仕様】
- 縦型（9:16）
- 15〜30秒
- シネマティックな映像、スローモーション多用
- BGMはエピック/ドラマチック系
- テロップは大きく太く、日本語
- 最後に「ツリスポ tsurispot.com」のテロップを2秒表示
```

## カテゴリ1: 釣り方How-to（保存される系）

### 1-1. サビキ釣りの仕掛けセット
```
釣り初心者向けの教育リール動画を作ってください。

テーマ：サビキ釣りの仕掛けセット方法

構成：
1.（冒頭テロップ）「サビキ、これだけ覚えればOK」
2. 竿にリールをセットする手元のアップ映像
3. 道糸にサビキ仕掛けを結ぶ手元のクローズアップ
4. コマセカゴにアミエビを詰める映像（「8分目がベスト」のテロップ）
5. 完成した仕掛けを海に投入する映像
6. アジが釣れてバケツに入れるシーン
7.（締めテロップ）「これだけで釣れます」「ツリスポ tsurispot.com」

雰囲気：明るい晴天の堤防、初心者でも安心感のある映像。手元のクローズアップ多め。
```

## ffmpegテキスト追加コマンド（必要な場合）

Gemini動画にブランドテロップだけ追加したい場合：
```bash
ffmpeg -y -i input.mp4 \
  -vf "drawtext=text='ツリスポ tsurispot.com':fontsize=28:fontcolor=white@0.6:x=60:y=h-80:shadowcolor=black@0.4:shadowx=2:shadowy=2" \
  -c:a copy output.mp4
```

縦型変換（横→縦、ぼかし背景）：
```bash
ffmpeg -y -i input.mp4 \
  -vf "split[bg][fg];[bg]scale=1080:1920:force_original_aspect_ratio=increase,crop=1080:1920,gblur=sigma=30,eq=brightness=-0.15[blurred];[fg]scale=1080:-2[scaled];[blurred][scaled]overlay=(W-w)/2:(H-h)/2" \
  -c:a copy output_vertical.mp4
```
